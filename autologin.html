This is a Userscript program written by David Feng in 2019, aiming to automate the process of logging into School Intranet.

This program can be modified to work with other Firefly Learning school VLEs as well.

Version: beta build 0.5 @ 2019/02/04 for Shrewsbury School Intranet

Update: New functions of Tasks Page "Mark as Done" button; other website autologin functions work in progress; Settings Page redesigned.

Note: This Userscript was debugged and tested in Chrome only, please switch to Chromium based browser for best experience.

Email me to give compliments, offer suggestions or report bugs

üîíSecurity

You should never install this extension on a public machine!

Any machine with this extension set up will be linked to that person's firefly.

If you install this extension on any public computer, not only that you caused inconvenience among others because they are constantly logged onto your account, your account login password may also be exposed to others.
